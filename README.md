## Project Overview

The **Student Management System (SMS)** project is designed to teach:

- Java OOP concepts in a real-world context  
- Spring Boot architecture and REST API development  
- Database integration using MySQL & JPA  
- Security with Spring Security & JWT  
- Testing, logging, and production-ready practices  

This project is ideal for **beginners to intermediate developers** aiming to build **full-featured Spring Boot applications**.
